# Engineering Decision Log

Project: Project Fawkes

---

## EDR-001

### Title

Use the Dell OptiPlex 7050 as the learning platform.

### Decision

The Dell OptiPlex 7050 will serve as the primary Linux learning workstation prior to restoring the 2008 Apple iMac.

### Rationale

- Lower project risk
- Modern hardware
- Better Linux compatibility
- Faster troubleshooting
- Allows the iMac restoration to become a deployment instead of an experiment

Status

Approved

---

## EDR-002

### Title

Separate operating system from engineering data.

### Decision

Deploy Linux on the 256 GB SSD.

Reserve the 1 TB drive for engineering projects, documentation, scripts, and backups after Linux installation has been validated.

Status

Approved
