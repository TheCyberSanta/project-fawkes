# System Overview

Project Fawkes is based on a 24-inch Apple iMac (Early 2008) that will be repurposed into a Linux Mint engineering workstation.

The system is currently running macOS El Capitan 10.11.6 and remains fully operational.

# Future Upgrade Considerations

- Increase RAM to 6–8 GB if cost effective.
- Evaluate replacement of aging cooling fans if noise or wear is observed.
- Consider a larger SSD if additional storage becomes necessary.

---

## Engineering Notes

The existing hard drive will be retained as a rollback option until Linux Mint has been fully validated on the replacement SSD.

The restoration will prioritize preservation of original Apple hardware wherever practical.
